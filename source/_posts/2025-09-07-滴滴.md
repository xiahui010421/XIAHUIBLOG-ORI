## 扫地机器人回到初始位置的最少修改次数
```
// 1. 接收指令输入（可替换为实际需求的输入方式，如页面输入框获取）
const input = prompt("请输入扫地机器人的指令（仅包含U、D、L、R）：") || "";

// 2. 统计各方向指令的数量
let up = 0, down = 0, left = 0, right = 0;
for (const char of input) {
  switch (char) {
    case "U": up++; break;
    case "D": down++; break;
    case "L": left++; break;
    case "R": right++; break;
  }
}

// 3. 计算各方向需修改的次数
// 垂直方向（U/D）：需让 up = down，修改次数 = 位移差的一半
const verticalModify = Math.floor(Math.abs(up - down) / 2);
// 水平方向（L/R）：需让 left = right，修改次数 = 位移差的一半
const horizontalModify = Math.floor(Math.abs(left - right) / 2);

// 4. 总最少修改次数
const minModifyCount = verticalModify + horizontalModify;

// 5. 输出结果
console.log(`最少需要修改的字符个数：${minModifyCount}`);
alert(`最少需要修改的字符个数：${minModifyCount}`);
```
##v使得尽可能多的市民能买到房子，从而让无家可归的市民数量最少。核心思路是贪心算法：让能负担低价房的市民优先购买低价房，这样能最大化利用补贴和市民的购房能力，留下高价房给有能力负担的市民，同时合理使用补贴。
```
const readline = require('readline');
const rl = readline.createInterface({
    input: process.stdin,
    output: process.stdout
});

rl.question('', (line1) => {
    // 1. 处理第一行输入：n(市民数)、m(房产数)、V(补贴总额)
    const [n, m, V] = line1.trim().split(' ').map(Number);
    rl.question('', (line2) => {
        // 2. 市民能负担的最大房价（升序：让能力弱的先选）
        const a = line2.trim().split(' ').map(Number).sort((x, y) => x - y);
        rl.question('', (line3) => {
            // 3. 房产价格（升序：让低价房先被选，最大化匹配数）
            const b = line3.trim().split(' ').map(Number).sort((x, y) => x - y);
            
            let i = 0; // 市民指针（从能力最弱开始）
            let j = 0; // 房产指针（从价格最低开始）
            let remainingSubsidy = V; // 剩余补贴（可拆分使用）

            while (i < n && j < m) {
                const needSubsidy = b[j] - a[i]; // 买房需要的补贴金额
                if (a[i] >= b[j]) {
                    // 情况1：市民直接买，不用补贴
                    i++;
                    j++;
                } else if (needSubsidy > 0 && needSubsidy <= remainingSubsidy) {
                    // 情况2：需要补贴，且剩余补贴够，用补贴买
                    remainingSubsidy -= needSubsidy; // 减去用掉的补贴
                    i++;
                    j++;
                } else {
                    // 情况3：没补贴/补贴不够，市民无家可归
                    i++;
                }
            }

            // 无家可归人数 = 总市民数 - 成功购房数（j是成功匹配的房产数）
            console.log(n - j);
            rl.close();
        });
    });
});
```
